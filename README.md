# Multiplication Table Generator (Java)

This is a simple Java program that generates the multiplication table of a number provided by the user. The program allows the user to decide how many multiples of the number should be displayed.

It demonstrates basic Java programming concepts such as user input, loops, and arithmetic operations.

## Features

* Accepts a number from the user
* Accepts the number of multiples to display
* Generates a multiplication table up to the specified count
* Uses clean console output for readability

## Concepts Used

* Java `Scanner` for user input
* `for` loop for iteration
* Basic arithmetic operations
* Console output formatting

## How the Program Works

1. The program asks the user to enter a number.
2. The user then enters the count up to which the table should be generated.
3. A loop runs from 1 to the given count.
4. Each iteration prints the multiplication result.

## Example Output

Enter a number: 5

Enter the count: 10


5 × 1 = 5

5 × 2 = 10

...
